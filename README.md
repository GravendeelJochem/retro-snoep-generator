# 🍬 Retro Snoep Generator

Een feestelijke tool voor teams die elke sprint iemand aanwijzen om snoep te regelen — inclusief een willekeurige suggestie voor het soort snoep.

🔗 **[Open de generator](https://gravendeeljochem.github.io/retro-snoep-generator/)**

---

## Wat doet het?

Tijdens de retrospectief druk je op **SPIN**. De generator kiest willekeurig:
- 🧑 **Wie** er de volgende sprint snoep meeneemt
- 🍫 **Wat** voor snoep dat moet zijn

Het resultaat verschijnt met een slot-machine animatie en confetti.

---

## Functies

- 🎰 Slot-machine animatie met vertragende spin
- 🎉 Confetti bij de onthulling
- 👥 Teamleden beheren — voeg toe of verwijder, geen maximum
- 🍫 Snoeplijst aanpassen — standaard 17 soorten, volledig bewerkbaar
- 📋 Geschiedenis van eerdere rondes
- 💾 Alles wordt lokaal opgeslagen in de browser (localStorage) — je hoeft het maar één keer in te stellen

---

## Gebruik

1. Open de [generator](https://gravendeeljochem.github.io/retro-snoep-generator/)
2. Ga naar **Teamleden** en vul de namen van je team in → Opslaan
3. Pas eventueel de **Snoeplijst** aan naar jullie smaak → Opslaan
4. Druk tijdens de retro op **SPIN** en geniet van de spanning

> De instellingen worden per browser opgeslagen. Gebruik je de generator op een gedeeld scherm, dan hoef je maar één keer in te stellen.

---

## Lokaal draaien

Geen installatie nodig. Download `index.html` en open het in je browser.

```bash
git clone https://github.com/GravendeelJochem/retro-snoep-generator.git
cd retro-snoep-generator
open index.html
```
